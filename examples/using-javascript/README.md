## Usage with global packages

# NPM:

```bash
$ npm install -g graphql-binding
$ graphql-binding --input schema.js --language javascript --outputBinding binding.js
node ./index.js
```

# Yarn:

```bash
$ yarn global add graphql-binding
$ graphql-binding --input schema.js --language javascript --outputBinding binding.js
node ./index.js
```

## Usage with package scripts

# NPM

```bash
$ npm run generate
$ npm start
```

# Yarn

```bash
$ yarn run generate
$ yarn start
```
